## 개요
- 영어단어 외우기 프로그램 제작
- 플랫폼 : PC
- 개발 도구 : JAVA 8 (swing, AWT), Eclipse
- 개발 내용 :
  - Swing, AWT 및 자바 GUI 프로그램 능숙화를 위한 미니 프로젝트 수행
  - 단어 등록 및 저장
  - 단어 로드 후 보기 / 쓰기 기능 이용해 영어단어 외우기
  - 등록 및 외우기 시에 엔터를 통한 커서 이동 등 편의사항 구현
  - 테스트, 모의고사 등을 통한 채점 시스템 구현 등
