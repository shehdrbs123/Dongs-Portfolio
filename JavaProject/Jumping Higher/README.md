## 개발 내용 
  - Swing, AWT를 이용한 GUI 구현
  - 상속을 이용해 4개의 발판 구현, 씬을 로드하여 사용할 발판 선택 가능
  - 입력, 그리기, 충돌 분리하고 멀티스레딩을 활용하여 반응성 향상
  - 시간 레코드 기능 구현
  - 추락 시 게임 종료, 메인 이동 구현
